# 🚀 Infinova.IO — Digital Agency Web App

Infinova is a modern, responsive digital agency web application built to showcase services, projects, and brand identity with a sleek UI, smooth animations, and an engaging user experience.

---

## 🌐 Live Preview
> https://infinova-io.vercel.app/

---

## ✨ Features

- ⚡ Fast and optimized with Vite
- 🎨 Beautiful UI using Tailwind CSS
- 🧩 Reusable components with shadcn/ui
- ✨ Smooth animations powered by Motion (motion.dev)
- 🎬 Interactive UI effects with Magic UI
- 📱 Fully responsive design
- 🌙 Dark / Light mode support
- 🧭 Smooth navigation and clean layout

---

## 🛠️ Tech Stack

- **Frontend:** React.js
- **Build Tool:** Vite
- **Styling:** Tailwind CSS
- **UI Components:** shadcn/ui
- **Animations:** Motion (motion.dev), Magic UI
